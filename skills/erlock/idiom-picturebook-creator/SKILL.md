# 中国成语绘本故事

该技能用于创作富有吸引力的绘本故事，通过图文结合的方式向儿童和学习者讲解中国成语，使内容更加易懂和有趣。

## 工作流程

1. 选择一个成语
    - 选择一个具有清晰故事背景或寓意的经典成语
2. 研究成语来源
    - 了解其历史背景和原始含义
3. 儿童化改编 - 简化语言表达，同时保留核心寓意
4. 生成关键故事场景（文本）
5. 设计主要故事角色（文本）
6. 为关键故事场景生成背景图
    - 使用 `txt2img_aly` 工具生成
    - 使用 `img2txt_aly` 工具检查
7. 生成主要角色形象图
    - 使用 `txt2img_aly` 工具生成
    - 使用 `img2txt_aly` 工具检查
8. 角色场景融合
    - 将步骤 6（背景图）与步骤 7（角色形象图）进行组合，生成每个关键场景的完整插画
    - 调整角色在场景中的比例、位置和互动关系，确保画面协调
    - 使用 `img2img_aly` 工具生成
    - 使用 `img2txt_aly` 工具检查
9. 绘本结构编排
    - 按绘本形式组织内容，将每一页的文字与对应插图配对
10. 故事连贯性审查
11. 生成最终绘本
12. 保存（可选）
    - 在根项目生成目录 `picturebook/<成语>`，里面有对应 md 文档（图片需要离线化）和 images （图片需要下载到这里）

## 何时使用该技能

- 用户请求“适合儿童的中国成语故事”
- 请求“带插图的成语解释”
- 需要将成语内容制作成“绘本形式”
- 希望获得关于某个成语的**图文结合**内容
