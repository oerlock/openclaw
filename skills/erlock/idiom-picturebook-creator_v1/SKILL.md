# 中国成语绘本故事

该技能用于创作富有吸引力的绘本故事，通过图文结合的方式向儿童和学习者讲解中国成语，使内容更加易懂和有趣。采用标准化故事板管理方法，确保故事结构完整、视觉风格统一。

## 工作流程

### 阶段1：故事规划
1. **选择成语** - 选择具有清晰故事背景或寓意的经典成语
2. **研究来源** - 了解历史背景和原始含义
3. **确定目标** - 明确目标年龄群体和核心寓意

### 阶段2：故事板创建
4. **生成故事板** - 使用 `scripts/generate_storyboard.py` 创建标准化12页故事结构
5. **填充内容** - 为每个页面编写场景描述和文字内容
6. **角色设计** - 定义主要和次要角色的特征

### 阶段3：视觉创作
7. **背景生成** - 为关键场景生成背景图
   - 使用 `txt2img_aly` 工具生成
   - 参考 `references/storyboard-template.md` 中的视觉风格指南
   - 使用 `img2txt_aly` 工具验证内容
8. **角色生成** - 创建主要角色形象图
   - 使用 `txt2img_aly` 工具生成
   - 确保角色风格与整体视觉风格一致
   - 使用 `img2txt_aly` 工具验证内容
9. **场景合成** - 将背景与角色融合
   - 使用 `img2img_aly` 工具进行图像编辑
   - 调整角色比例、位置和互动关系
   - 使用 `img2txt_aly` 工具验证最终效果

### 阶段4：完成与输出
10. **连贯性审查** - 确保故事流畅性和视觉一致性
11. **最终编排** - 按标准绘本格式组织内容
12. **保存输出** - 在 `picturebook/<成语>/` 目录中保存完整绘本

## 核心资源

### 脚本工具
- **`scripts/generate_storyboard.py`** - 自动生成标准化12页故事板结构
- **`scripts/process_images.py`** - 处理图像提示生成和故事板验证

### 参考资料
- **`references/storyboard-template.md`** - 详细的故事板模板和创作指南

## 视觉规格
- **图像尺寸**: 1664*928 (横向) 或 928*1664 (纵向)
- **文本限制**: 每页不超过50字，适合儿童阅读
- **艺术风格**: 明亮卡通风格，融入中国传统元素
- **色彩调性**: 温暖明亮的色彩搭配

## 何时使用该技能

- 用户请求"适合儿童的中国成语故事"
- 请求"带插图的成语解释"
- 需要将成语内容制作成"绘本形式"
- 希望获得关于某个成语的**图文结合**内容
- 需要标准化的故事板管理流程
