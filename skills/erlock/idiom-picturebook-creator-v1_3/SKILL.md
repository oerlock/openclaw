# 中国成语绘本故事

该技能用于创作富有吸引力的绘本故事，通过图文结合的方式向儿童和学习者讲解中国成语，使内容更加易懂和有趣。

## 工作流程

在根项目生成目录 `picturebook/<故事名：英文>`，随后流程的中间内容放在 `storyboard.md` 中。

1. 选择一个成语
    - 选择一个具有清晰故事背景或寓意的经典成语
2. 研究成语来源
    - 了解其历史背景和原始含义
3. 儿童化改编 - 简化语言表达，同时保留核心寓意
4. 生成关键故事场景（文本）
5. 设计主要故事角色（文本）
6. 为关键故事场景配图
   - 根据关键故事场景描述使用 `txt2img_aly` 生成场景配图
   - 生成的结果更新到 `storyboard.md`
7. 生成主要角色形象图
   - 根据主要角色设计使用 `txt2img_aly` 生成场景配图
   - 生成的结果更新到 `storyboard.md`
8. 角色场景融合
    - 遍历关键故事场景，使用 `img2img_aly` 将对应的角色形象与对应的场景配图结合，生成每个关键场景的完整插画
9. 绘本结构编排
    - 按绘本形式组织内容，将每一页的文字与对应插图配对
10. 生成最终绘本

## 何时使用该技能

- 用户请求“适合儿童的中国成语故事”
- 请求“带插图的成语解释”
- 需要将成语内容制作成“绘本形式”
- 希望获得关于某个成语的**图文结合**内容
