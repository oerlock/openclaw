# 成语故事讲述者

本技能将中国成语及其历史典故转化为适合儿童阅读的生动故事，篇幅约800字，并支持对已生成故事的修改和优化。

## 工作流程

1. **识别成语**：从用户请求中提取目标成语
2. **研究典故**：理解成语的历史背景、人物和道德寓意
3. **儿童化改编**：简化复杂概念，添加贴近儿童生活的元素，同时保留核心寓意
4. **故事结构**：创作有清晰开头、发展和结尾的叙事，适合6-12岁儿童
5. **保持文化准确性**：在确保可读性的同时，忠实传达成语的本质含义
6. **故事修改**：根据用户反馈对已生成的故事进行调整和优化，包括语言风格、情节细节、角色塑造等方面的修改

## 故事创作指南

- **篇幅**：约800汉字
- **语言**：使用简单、生动的中文，符合儿童阅读水平
- **人物塑造**：赋予历史人物适合儿童理解的性格特点，同时尊重其本质
- **道德寓意**：清晰传达成语的教育意义，避免说教
- **趣味性**：包含对话、感官细节和情感元素，增强故事吸引力
- **文化背景**：在故事中自然融入对陌生历史概念的简要解释

## 输出格式

以完整叙事形式呈现故事，结尾处标注：
- **成语**：[原始成语]
- **寓意**：[用简单语言解释的道德寓意]

直接输出故事内容，无需包含关于改编过程的说明性文字。

## 产出位置

生成的故事文件应保存在工作区根目录的以下路径：
```
output/txt/idiom_story_for_child/<成语名称>.md
```

例如，"画龙点睛"的故事应保存为：
```
output/txt/idiom_story_for_child/画龙点睛.md
```

修改后的版本会覆盖原文件，确保始终保存最新版本的故事内容。
