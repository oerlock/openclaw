# 故事修改指南

## 常见修改类型

### 1. 语言难度调整
- **降低难度**：使用更简单的词汇，缩短句子，增加重复结构
- **提高难度**：适当引入成语、典故相关词汇，增加描述性语言
- **年龄适配**：6-8岁用更口语化表达，9-12岁可适当增加文学性

### 2. 情节优化
- **增强冲突**：让问题更具体，解决方案更清晰
- **简化情节**：删除次要人物或支线，聚焦核心故事
- **增加趣味**：添加幽默元素、拟人化描述或互动场景

### 3. 文化内容调整
- **背景现代化**：将古代场景转换为现代儿童熟悉的环境
- **保留传统**：确保核心文化元素和道德寓意不变
- **增加解释**：对复杂的历史概念提供更详细的说明

### 4. 教育目标强化
- **突出寓意**：让道德教训更自然地融入故事结尾
- **增加互动**：在故事中设置思考问题或讨论点
- **联系现实**：帮助儿童理解成语在现代生活中的应用

## 修改流程

1. **理解需求**：明确用户希望修改的具体方面
2. **保留核心**：确保成语的核心寓意和关键典故元素不变
3. **渐进调整**：一次专注于一个方面的修改，避免过度改动
4. **保持一致性**：修改后的故事风格应与原文保持一致
5. **验证效果**：确保修改后的故事仍然适合目标年龄段

## 注意事项

- 避免过度现代化而失去传统文化精髓
- 不要为了趣味性而扭曲成语的原始含义
- 保持故事的教育价值和文化传承功能
- 确保修改后的版本仍然控制在800字左右
